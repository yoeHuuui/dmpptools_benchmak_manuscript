# dmpptools Manuscript Benchmark and Plot Code

This repository contains the benchmark and plotting code used for the dmpptools manuscript.

The repository is organized into two main parts:

* `manuscript_benchmark/`: code for running benchmark experiments in manuscript.
* `manuscript_plot/`: code for generating manuscript and supplementary figures.

Generated figure files, such as PDF outputs, are not included in this repository. They can be regenerated by running the corresponding plotting scripts.


## Requirements


Additional packages may be required depending on individual benchmark or plotting scripts. Please check the import statements in each script if a missing-package error occurs.

## Usage

Clone the repository:

```bash
git clone https://github.com/yoeHuuui/dmpptools_benchmak_manuscript.git
cd dmpptools_benchmak_manuscript
```

### Benchmark code

Benchmark scripts are located in the `benchmark/` directory.

```bash
cd manuscript_benchmark
```

Run the corresponding benchmark script according to the specific experiment.

### Plot code

Plotting scripts are located in the `manuscript_plot` directory.

For example:

```bash
cd manuscript_plot/fig_1
Rscript code/fig_1.R
```

## Notes

* `manuscript_benchmark/` contains benchmark-related code and experiment scripts.
* `manuscript_plot/` contains figure-generation code for the manuscript and supplementary materials.
* Large datasets, intermediate files, and generated outputs should not be committed unless required for reproducibility.
* If external data files are needed, please describe their source and expected location in the relevant subdirectory.
